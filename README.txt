# TeamProject #1 - 후방 감지 센서 기반 GUI 제어 시스템

![Project Banner](banner.png) <!-- 필요 시 프로젝트 배너 이미지 -->

## 프로젝트 개요
본 프로젝트는 **후방 감지 센서**를 활용하여 **아두이노 3개**와 **PyQt5 기반 GUI**를 통해 차량 바퀴를 제어하는 시스템입니다.

- **목적:** 실시간 거리 감지 및 모터 제어를 GUI에서 직관적으로 수행
- **주요 기능:** 거리 표시, LED 경고, 부저 알림, 모터 제어(CW, CCW, 좌/우 이동, 정지)

---

## 시스템 구성
| 구성 요소 | 역할 |
|------------|------|
| **아두이노 1 (Master)** | 초음파 센서, LED, LCD, 부저 제어 및 슬레이브 아두이노 2개 통신 |
| **아두이노 2, 3 (Slave)** | 각각 왼쪽/오른쪽 DC 모터 제어 |
| **PyQt5 GUI** | 센서 데이터 시각화, 모터 제어 버튼 제공 |
| **통신 방식** | Master ↔ Slave: I2C, Master ↔ PC: Serial |

---

## 아두이노 제어 구조
- Master 아두이노가 슬레이브 아두이노에 명령 전달
- 각 슬레이브 아두이노는 모터의 방향 및 속도(PWM) 제어
- Master 아두이노는 초음파 센서로 거리 측정 후 GUI에 전달
- 거리와 경고 수준에 따라 LED 색상 변경 및 부저 알림

---

## GUI 기능 (PyQt5)
### 거리 표시
- 초음파 센서로 측정한 거리 실시간 표시

### 모터 제어 버튼
| 버튼 | 설명 |
|------|------|
| CW | 모터 시계 방향 회전 |
| CCW | 모터 반시계 방향 회전 |
| LEFT | 좌측 이동 |
| RIGHT | 우측 이동 |
| STOP | 모터 정지 |
| BUZZER | 경고음 작동 |

### LED 상태 표시
- 초록 (`g.png`) → 안전  
- 노랑 (`y.png`) → 주의  
- 빨강 (`r.png`) → 경고  

![LED Status](g.png) ![LED Status](y.png) ![LED Status](r.png)

## GUI 데이터 처리
- Serial 통신으로 Master 아두이노와 연결
- 수신 메시지 처리:
  - `DISTANCE : xx` → 거리 표시
  - `DATA : status` → 모터 상태 출력
  - `ALERT : [G/Y/R]` → LED 색상 변경
